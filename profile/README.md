syssec-debugging-tools is a collection of tools for debugging in system security research including (but not limited to) deep program manipulation, development of side-channel analysis tools, building hardening or verification frameworks, or finding attack vectors.
The organization serves as a coordination point for standalone tools that are independently maintained and provides structure and curation.
